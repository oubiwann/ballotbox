Introduction¶

About ballotbox¶

ballotbox is a Python library that attempts to implement useful voting methodologies.

This project started as a result of conversations and experimentation that took place on the ULS-SIG when discussing self-organizing object meshes.

Dependencies¶

The current implementation depends upon the following additional software and/or libraries:

  • Python
  • zope.interface
  • sphinx
  • repoze.sphinx.autointerface
